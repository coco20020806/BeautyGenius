# 化妆步骤 taxonomy（视频解析切分参考）

版本：v1  
用途：解析美妆教程视频时，**步骤划分与命名**应优先对齐本文档的一级/二级类目；口播、字幕、画面文字中出现的内容尽量归入对应细分项。

## 使用说明（给解析模型 / Agent）

1. **一级步骤 `step_name`**：从下表 12 个主类中选择（或组合教程中实际出现的子集），按视频时间顺序排列。
2. **二级 `sub_steps` / 标签**：记录本段涉及的具体部位、技法或问题域（可多选）。
3. 主类标注 **「不做」**：教程明确跳过或未出现时，可不生成该步骤，或在 step 的 `notes` 中标记 `skipped`。
4. 同一段口播若跨多个主类，以**当前正在操作的产品/部位**决定归属；边界以博主换产品或换部位为准。
5. 输出 JSON 时建议在每步增加可选字段 `taxonomy`（见文末示例），便于下游检索；若 schema 尚未扩展，可将细分项写入 `step_name` 后缀或 `text.on_screen` 归纳句。

## 1. 妆前

**细分步骤**

| 类目 | 说明 |
|------|------|
| 局部打底 | 分区预处理，非全脸底妆 |

**局部打底 — 关注维度**

- 使用区域  
- 用量  
- 等待时间  
- 是否需要避开某些区域  

---

## 2. 底妆

**细分步骤 / 技法维度**

| 类目 | 说明 |
|------|------|
| 内轮廓 | 面中、T 区等内圈上妆范围 |
| 外轮廓 | 面颊外侧、发际附近衔接区 |
| 薄涂 | 少量、低遮盖、轻薄铺色 |
| 厚涂 | 叠加、提高遮盖或饱和度 |

---

## 3. 遮瑕

- 黑眼圈  
- 泪沟  
- 痘印  
- 泛红  
- 鼻翼  
- 嘴角  
- 局部斑点  

---

## 4. 定妆

按**区域**细分（可多选）：

- 额头  
- 鼻部  
- 眼下  
- 鼻翼  
- 嘴角  
- 下巴  

---

## 5. 眉毛

**默认 taxonomy**：**不做**（教程未涉及眉妆时可跳过本步；若涉及则改用下方任意实际技法并去掉「不做」标记）

常见可扩展细分（教程出现时选用）：眉峰 / 眉尾 / 眉头 / 毛流 / 染眉等（本 v1 未强制枚举）。

## 建议写入 analysis 的扩展字段（可选）

在既有 `steps[]` 条目上增加（不改变现有必填字段）：

```json
{
  "step_index": 6,
  "step_name": "眼睛",
  "taxonomy": {
    "primary": "眼睛",
    "sub_steps": ["上眼尾", "外V区", "卧蚕-中段"],
    "skipped": false
  },
  "time_range": { "start_sec": 120, "end_sec": 180, "start_label": "2:00", "end_label": "3:00" }
}
```

眉毛 / 睫毛整步跳过时：

```json
{
  "step_name": "眉毛",
  "taxonomy": { "primary": "眉毛", "sub_steps": [], "skipped": true }
}
```
